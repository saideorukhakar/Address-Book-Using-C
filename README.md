# Address-Book-Using-C
The Address Book Project is a console-based C application that manages contacts by storing names, phone numbers, and email addresses. It allows users to add, search, update, and delete contact details using standard input and output.

Description :
-----------------------------------------------------------------------------------------------------------------
The Address Book Project is a console-based C application that manages contacts by storing names, phone numbers, 
and email addresses. It allows users to add, search, update, and delete contact details using standard input and 
output, which is stored persistently in a CSV file.
